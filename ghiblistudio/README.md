# Ghibli studio App

La idea del proyecto es utilizar los endpoints proporcionados por la api de [studio ghibli](https://ghibliapi.herokuapp.com/#)

<ul>
  <li> Films </li>
  <li> People </li>
  <li> Locations </li>
  <li> Species </li>
  <li> Vehicles </li>
</ul>

## Estructurado
